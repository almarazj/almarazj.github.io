Modificación de la escala temporal de audio

En este informe se detalla la implementación de un algoritmo en python diseñado para escalar temporal mente una señal de audio. El mismo está basado en las técnicas Overlap and Add para las componentes percusivas de la señal y Phase Vocoder para las componentes armónicas. Se implementa un filtro de mediana movil para la separación de los componentes de la señal y luego de su procesado se suman para obtener la señal escalada. Además, se analizan distintos algoritmos existentes para luego compararlos con los resultados obtenidos en el algoritmo implementado en este trabajo.
